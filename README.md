# NYTimes App for Faifly


## Frameworks used
```
Alamofire
```
Alamofire был использован для удобной и более комфортной работы с запросами

```
SwiftyJSON
```
SwiftyJSON был использован для работы с JSON, получаемого в ответ на запрос к Article Search API
